    -  Estenda o código para múltiplas dimensões (vetor de parâmetros).
    -  Implemente o algoritmo de \textit{Gibbs Sampling} para uma distribuição normal bivariada.

    -  Verifique como a taxa de aceitação varia com a largura da proposta (`sigma\_prop`).
    -  Como o Fortran é utilizado em projetos de larga escala na cosmologia (ex: CosmoMC)?
