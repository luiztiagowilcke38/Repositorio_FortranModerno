# Resolução: Séries Temporais em Alta Frequência (Tick Data)

## Exercícios do Livro

    -  Derive a distribuição dos tempos de espera entre eventos sob um processo de Poisson.
    -  Explique o viés de assinatura (\textit{signature plot}) na volatilidade realizada.
    -  Derive o estimador TSRV (\textit{Two-Scale Realized Volatility}).
    -  Desenvolva uma estrutura de dados de Livro de Ofertas (\textit{Limit Order Book}) dinâmica.
    -  Implemente um filtro para remover \textit{outliers} e erros de gravação em dados tick.
    -  Analise a sazonalidade intradiária da volatilidade (Padrão-U).
    -  Estude a correlação entre volume transacionado e variação de preço.
    -  Investigue o impacto de algoritmos de HFT (\textit{High-Frequency Trading}) na liquidez do mercado.
