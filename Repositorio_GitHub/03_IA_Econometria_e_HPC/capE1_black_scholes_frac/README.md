# Resolução: Econometria de Caudas Longas: O Modelo de Black-Scholes Fracionário

## Exercícios do Livro

    -  Mostre que o modelo de Black-Scholes fracionário não admite arbitragem sob certas condições de integração.
    -  Explique a relação entre o expoente de Hurst $H$ e a ordem da derivada fracionária $\alpha$.
    -  Use a SuperDiFF para comparar o preço de uma opção Europeia com $\alpha=1$ e $\alpha=0.7$.
    -  Verifique como o sorriso de volatilidade (volatility smile) é afetado pela fracionaridade.
    -  Desenvolva uma estratégia de hedging delta-neutra para este modelo.
