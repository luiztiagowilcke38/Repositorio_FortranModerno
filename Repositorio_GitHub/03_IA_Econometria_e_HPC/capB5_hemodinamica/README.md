# Resolução: Biofísica Cardiovascular: Hemodinâmica e Elasticidade de Vasos

## Exercícios do Livro

    -  Derive a fórmula de Moens-Korteweg para a velocidade da onda de pulso.
    -  Discuta o efeito da não-newtonianidade do sangue em pequenos vasos (Modelo de Casson).
    -  Estenda o modelo para uma rede arterial ramificada (Árvore de Bifurcações).
    -  Investigue a formação de estenoses e como elas alteram o perfil de pressão local.
    -  Use a SuperDiFF para calcular o regime de fluxo reverso na aorta descendente durante a diástole.
