# Resolução: Redes Neurais Físicas em Fortran

## Exercícios do Livro

    -  Derive a regra da cadeia para uma rede de 3 camadas.
    -  Explique por que a função de ativação ReLU evita o problema de "vanishing gradient".
    -  Derive o gradiente do funcional de perda físico para a equação de Poisson.
    -  Escreva a subrotina para a função de ativação `sigmoid` e sua derivada.
    -  Implemente a inicialização de Xavier/He para os pesos.
    -  Treine uma rede para aprender o potencial do oscilador harmônico.
    -  Compare o tempo de treinamento em Fortran vs frameworks como TensorFlow.
    -  Investigue como as redes neurais podem ser usadas para acelerar simulações de DFT.
