# Resolução: Redes Neurais Biológicas: O Modelo de Hodgkin-Huxley

## Exercícios do Livro

    -  Derive a base física para a dependência da quarta potência de $n$ na corrente de Potássio.
    -  Explique o conceito de período refratário absoluto e relativo.
    -  Simule a resposta de um neurônio a uma corrente constante e plote o trem de disparos (\textit{spiking}).
    -  Implemente o modelo simplificado Izhikevich para acelerar simulações de redes massivas.
    -  Determine o limiar de corrente necessário para disparar um potencial de ação.
    -  Estude a sincronização de dois neurônios acoplados eletricamente via \textit{gap junctions}.
    -  Como o projeto \textit{Blue Brain} utiliza supercomputadores para simular colunas corticais completas?
