# Resolução: Dinâmica Molecular e o Problema do Dobramento de Proteínas

## Exercícios do Livro

    -  Demonstre que o integrador de Langevin recupera a distribuição de Boltzmann no limite de tempo longo.
    -  Derive o raio de giração ($R_g$) para uma cadeia polimérica ideal.
    -  Implemente o potencial de "WCA" (Weeks-Chandler-Andersen) na sub-rotina de drift.
    -  Investigue como o tempo de dobramento escala com o coeficiente de fricção $\gamma$ (Kramers' turn-over).
    -  Use a SuperDiFF para calcular a Energia Livre ao longo da coordenada de reação (distância de ponta a ponta).
