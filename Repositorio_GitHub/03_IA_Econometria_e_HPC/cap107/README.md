# Resolução: Motor de Trading Quantitativo de Alta Frequência

## Exercícios do Livro

    -  O que é o \textit{Bid-Ask Spread} e como ele quantifica a liquidez de um ativo?
    -  Explique o conceito de latência de "ida e volta" (\textit{round-trip}) em sistemas de HFT.
    -  Desenvolva uma estrutura de dados de árvore rubro-negra para manter o LOB ordenado por preço.
    -  Implemente uma estratégia simples de "Mean Reversion" e teste-a contra dados históricos de ticks.
    -  Verifique como o impacto no mercado (\textit{Market Impact}) aumenta conforme o volume da ordem executada aumenta.
    -  Simule o fenômeno de \textit{Flash Crash} devido ao desequilíbrio abrupto entre ordens.
    -  Como o Fortran é integrado com bibliotecas de rede (\textit{ZeroMQ/Sockets}) para receber feeds de dados em tempo real?
