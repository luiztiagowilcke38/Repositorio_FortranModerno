# Resolução: Heterocedasticidade Condicional: Modelos GARCH e EGARCH

## Exercícios do Livro

    -  Derive a expressão para a curtose (kurtosis) de um processo GARCH(1,1) e mostre que ela é maior que 3.
    -  Explique por que o EGARCH utiliza o logaritmo da variância em vez da variância pura.
    -  Modifique o código para suportar o modelo GJR-GARCH.
    -  Utilize a SuperDiFF para prever o VaR (Value-at-Risk) de 1 dia para o índice IBOVESPA.
    -  Verifique se os resíduos padronizados do modelo estimado apresentam autocorrelação residual via teste de Ljung-Box.
