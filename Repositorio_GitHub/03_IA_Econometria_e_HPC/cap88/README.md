# Resolução: Dinâmica Molecular de Proteínas

## Exercícios do Livro

    -  Por que a Dinâmica Molecular clássica é válida para proteínas, apesar dos efeitos quânticos nos elétrons?
    -  Derive o integrador de Velocidade de Verlet a partir da expansão de Taylor.
    -  Implemente o algoritmo de \textit{Neighbor List} (Verlet List) para acelerar o cálculo de forças não-ligadas.
    -  Codifique um termostato simples (Andersen ou Berendsen) para manter a temperatura do sistema constante.
    -  Simule o "colapso hidrofóbico" de uma pequena cadeia de 10 polímeros em água.
    -  Estude a transição de fase de um aglomerado de 50 átomos de Argônio variando a energia total.
    -  Como supercomputadores dedicados como o `ANTON` conseguem atingir escalas de milissegundos em simulações de proteínas?
