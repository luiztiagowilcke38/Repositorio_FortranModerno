# Resolução: Equações Diferenciais Estocásticas (SDE)

## Exercícios do Livro

    -  Explique a diferença entre a interpretação de Itô e de Stratonovich para integrais estocásticas.
    -  Derive a Equação de Fokker-Planck associada a uma SDE simples e descreva sua interpretação física.
    -  Simule a evolução do preço de um ativo financeiro usando o Movimento Browniano Geométrico.
    -  Implemente o integrador de Milstein e compare sua precisão com o Euler-Maruyama.
    -  Verifique se a distribuição de velocidades de uma partíca Browniana converge para a distribuição de Maxwell-Boltzmann.
    -  Estude a captura de uma partícula em um poço de potencial ativado termicamente (Problema de Kramers).
    -  Como as SDEs são usadas na Biologia de Sistemas para modelar o ruído na expressão gênica dentro de células individuais?
