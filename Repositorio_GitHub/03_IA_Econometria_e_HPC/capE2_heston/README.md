# Resolução: Modelagem de Volatilidade Estocástica: O Modelo de Heston

## Exercícios do Livro

    -  Derive a EDP de Heston utilizando o Lema de Itô bi-dimensional.
    -  Discuta a condição de Feller $2\kappa\theta > \sigma^2$ e sua importância para a variância.
    -  Implemente uma versão que utilize o esquema de Milstein para maior precisão na variância.
    -  Plote a distribuição dos preços finais e compare com a Log-Normal de Black-Scholes.
    -  Calcule a superfície de volatilidade implícita gerada pelo modelo de Heston.
