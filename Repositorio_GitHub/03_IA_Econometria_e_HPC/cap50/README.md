# Resolução: Equilíbrio Geral Dinâmico Estocástico (DSGE)

## Exercícios do Livro

    -  Derive as condições de primeira ordem de um modelo RBC (\textit{Real Business Cycle}) básico.
    -  Explique a condição de Blanchard-Kahn para a unicidade da solução estável.
    -  Derive a Curva de Phillips Novo-Keynesiana linearizada.
    -  Desenvolva a rotina de log-linearização automática para uma função de utilidade CRRA.
    -  Integre a chamada à `DGGES` do LAPACK para resolver um sistema $2 \times 2$.
    -  Calcule as Funções de Resposta a Impulso (IRF) para um choque tecnológico.
    -  Verifique a sensibilidade dos resultados em relação ao parâmetro de persistência do choque $\rho$.
    -  Estude modelos DSGE com agentes heterogêneos (HANK) e as limitações do agente representativo.
