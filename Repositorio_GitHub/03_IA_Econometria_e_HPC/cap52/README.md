# Resolução: Teoria dos Jogos e Agentes em Fortran

## Exercícios do Livro

    -  Prove o Teorema de Nash para jogos finitos.
    -  Derive os pontos fixos da dinâmica de replicador para o jogo Falcão-Pomba (\textit{Hawk-Dove}).
    -  Explique o conceito de Estabilidade Evolutiva (ESS).
    -  Desenvolva um simulador de agentes para o modelo de segregação de Schelling.
    -  Implemente o algoritmo de Monte Carlo para evolução de estratégias em uma rede \textit{Scale-free}.
    -  Verifique a emergência de cooperação no dilema do prisioneiro iterado espacial.
    -  Estude a transição de fase entre estados de coordenação e desordem.
    -  Investigue o uso de ABM para a modelagem de epidemias (modelo SEIR espacial) e políticas públicas.
