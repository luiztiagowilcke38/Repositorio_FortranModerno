# Resolução: Wavelets e Decomposição Multirresolução

## Exercícios do Livro

    -  Prove que a wavelet de Haar é ortogonal.
    -  Explique a relação entre a Transformada Wavelet e o Princípio de Incerteza de Heisenberg.
    -  Estenda o solver de Haar para múltiplos níveis de decomposição (Pyramid Algorithm).
    -  Implemente o \textit{Thresholding} (SURE ou VisuShrink) para redução de ruído (denoising).
    -  Compare a compressão de um sinal de ECG usando Fourier (eliminando altas frequências) vs Wavelet (eliminando pequenos coeficientes).
    -  Como wavelets são utilizadas na detecção de transientes em sistemas de energia elétrica?
