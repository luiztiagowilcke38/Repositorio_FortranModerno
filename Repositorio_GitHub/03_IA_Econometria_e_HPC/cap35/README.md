# Resolução: Computação Quântica Simulada em Fortran

## Exercícios do Livro

    -  Prove que a porta Hadamard é unitária e sua própria inversa.
    -  Mostre que o estado de Bell $|\Phi^+\rangle = \frac{|00\rangle + |11\rangle}{\sqrt{2}}$ é emaranhado.
    -  Derive a matriz da porta CNOT para 2 qubits.
    -  Escreva o código Fortran para o produto tensorial de duas matrizes quânticas.
    -  Implemente a porta Toffoli (CCNOT).
    -  Verifique a unitariedade do estado após 1000 aplicações de portas aleatórias.
    -  Simule o Algoritmo de Grover para $n=3$ qubits.
    -  Investigue como o algoritmo de Shor fatora números através da QFT.
