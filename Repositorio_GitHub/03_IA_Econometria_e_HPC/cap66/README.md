# Resolução: Metaprogramação e Geração Automática de Código

## Exercícios do Livro

    -  Explique a diferença entre metaprogramação em tempo de compilação (Macros) e em tempo de execução (Reflection, não suportado nativamente no Fortran).
    -  Quais são os riscos de segurança e manutenção ao utilizar macros complexas do pré-processador C em Fortran?
    -  Escreva um script Python simples que gere uma sub-rotina de multiplicação de matrizes desenrolada (\textit{unrolled}) com fator $K$.
    -  Crie um sistema de arquivos `.inc` que permita instanciar o mesmo algoritmo para tipos `REAL` e `COMPLEX`.
    -  Compare o tempo de compilação de um código altamente "templatizado" via macros versus código manual.
    -  O que são as clausulas `SELECT TYPE` e `CLASS` do Fortran 2003 e como elas ajudam no polimorfismo (metaprogramação OO)?
    -  Investigue o projeto \textit{fypp} (Python-powered Fortran Preprocessor) e como ele é utilizado na biblioteca \textit{stdlib} do Fortran.
