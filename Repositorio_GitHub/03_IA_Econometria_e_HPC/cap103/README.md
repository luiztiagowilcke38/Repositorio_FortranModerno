# Resolução: Grupos de Lie e Simetrias em Física

## Exercícios do Livro

    -  Mostre que as constantes de estrutura do grupo SU(2) são dadas pelo símbolo de Levi-Civita $\epsilon_{ijk}$.
    -  Explique a diferença entre uma simetria global e uma simetria de calibre (\textit{gauge}).
    -  Desenvolva uma rotina para gerar as matrizes de Gell-Mann (geradores do grupo SU(3)).
    -  Verifique numericamente as relações de comutação do grupo SO(3) de rotações.
    -  Estude a representação adjunta de um grupo de Lie simples.
    -  Implemente a exponencial de uma matriz da álgebra de Lie para obter o elemento correspondente do grupo.
    -  Como o Fortran é usado em Física de Altas Energias para calcular amplitudes de espalhamento usando a álgebra de Dirac?
