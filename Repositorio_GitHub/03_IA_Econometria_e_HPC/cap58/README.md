# Resolução: Entropia e Teoria da Informação

## Exercícios do Livro

    -  Mostre que a entropia é máxima para uma distribuição uniforme.
    -  Prove que $I(X; Y) = H(X) + H(Y) - H(X, Y)$.
    -  Escreva uma sub-rotina que estime a distribuição de probabilidade (\textit{Histogramming}) de um sinal contínuo para calcular sua entropia.
    -  Implemente o cálculo da Divergência de Kullback-Leibler entre duas distribuições Gaussianas.
    -  Utilize a entropia de Shannon para medir a complexidade de diferentes cadeias de DNA fornecidas.
    -  Investigue como a entropia de permutação (\textit{Permutation Entropy}) pode ser usada para detectar caos em séries temporais.
    -  Como o Demônio de Maxwell se relaciona com o apagamento de informação (Princípio de Landauer)?
