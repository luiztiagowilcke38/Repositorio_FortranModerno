# Resolução: Dinâmica Multivariada: Modelos VAR e VECM

## Exercícios do Livro

    -  Derive a decomposição de Granger para um sistema $VAR(1)$ bidimensional.
    -  Explique por que o posto (rank) da matriz $\Pi$ determina o número de relações de cointegração.
    -  Estenda o código para calcular a Função de Resposta ao Impulso Ortogonalizada (OIRF).
    -  Utilize dados de inflação (IPCA) e juros (SELIC) para estimar um VECM e discuta o canal de transmissão monetária.
    -  Verifique a estabilidade do sistema VAR checando se os autovalores da matriz companheira estão dentro do círculo unitário.
