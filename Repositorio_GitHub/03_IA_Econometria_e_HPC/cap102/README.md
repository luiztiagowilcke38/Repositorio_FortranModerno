# Resolução: Visão Computacional de Alto Desempenho

## Exercícios do Livro

    -  Por que o preenchimento (\textit{padding}) é necessário nas bordas de uma imagem durante a convolução?
    -  Qual a diferença física e funcional entre um filtro de Gauss (suavização) e um filtro de Sobel (derivação)?
    -  Desenvolva um filtro de borramento Gaussiano usando um kernel 5x5 separável.
    -  Otimize a rotina de convolução utilizando a Transformada de Fourier (Teorema da Convolução).
    -  Verifique o efeito do ruído "sal e pimenta" na precisão da detecção de bordas.
    -  Implemente uma camada de \textit{Max-Pooling} para reduzir a resolução da imagem preservando características.
    -  Como o Fortran é usado no processamento de imagens do Telescópio Espacial James Webb para calibração de sensores?
