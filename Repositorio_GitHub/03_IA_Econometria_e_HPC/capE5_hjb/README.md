# Resolução: Teoria de Controle Ótimo e Equações de Hamilton-Jacobi-Bellman

## Exercícios do Livro

    -  Deduza a equação HJB utilizando o princípio da otimalidade de Bellman no limite de tempo contínuo.
    -  Verifique que para a utilidade $\ln(c)$, a taxa de consumo ótima é proporcional à riqueza.
    -  Use a SuperDiFF para resolver o problema de consumo e investimento com utilidade Power.
    -  Investigue como a introdução de uma taxa de juros estocástica altera a estrutura da equação HJB.
    -  Aplique o controle ótimo para resolver um problema de extração exaurível de recursos naturais (Modelo de Hotelling).
