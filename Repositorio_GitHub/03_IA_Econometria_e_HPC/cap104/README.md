# Resolução: Integrais de Caminho (Feynman Path Integrals)

## Exercícios do Livro

    -  Derive a Equação de Schrödinger a partir do propagador de Feynman para intervalos infinitesimais.
    -  Explique por que a trajetória clássica emerge da integral de caminho no limite $\hbar \to 0$ (limite estacionário).
    -  Desenvolva um código PIMC para calcular a densidade de probabilidade do estado fundamental do oscilador harmônico.
    -  Estude o comportamento de um potencial de poço duplo e observe o \textit{tunneling} quântico via integrais de caminho.
    -  Verifique a convergência dos resultados em função do número de \textit{beads} (passos temporais imaginários).
    -  Calcule a energia média do sistema usando o estimador virial.
    -  Como as integrais de caminho são usadas em Química Quântica para modelar efeitos de tunelamento em reações enzimáticas?
