# Resolução: Metabolismo Dinâmico: Cinética Enzimática com Atraso (DDE)

## Exercícios do Livro

    -  Realize a análise de estabilidade linear e encontre o valor crítico de $\tau$ para a bifurcação de Hopf.
    -  Explique por que o espaço de fase de uma DDE é infinito-dimensional.
    -  Estenda o código para incluir atrasos distribuídos (Kernel de Erlang).
    -  Use a SuperDiFF para mapear o regime de caos no parâmetro $n$ de Hill.
    -  Verifique a preservação da positividade das concentrações para grandes atrasos químicos.
