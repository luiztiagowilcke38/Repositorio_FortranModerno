# Resolução: Atratores Estranhos e Sistemas Caóticos

## Exercícios do Livro

    -  Mostre que o sistema de Lorenz é dissipativo, ou seja, o volume no espaço de fase contrai com o tempo.
    -  O que diferencia um atrator periódico (ciclo limite) de um atrator estranho?
    -  Integre o sistema de Lorenz por 10.000 passos e plote a projeção $X-Z$ (as famosas "asas da borboleta").
    -  Implemente o sistema de Rössler e explore suas diferentes rotas para o caos.
    -  Verifique a sensibilidade às condições iniciais integrando duas trajetórias com uma diferença de $10^{-10}$ em $X_0$.
    -  Construa um Mapa de Poincaré para o sistema de Lorenz e identifique a estrutura fractal.
    -  Como a teoria do caos impacta a previsão numérica do tempo em horizontes superiores a duas semanas?
