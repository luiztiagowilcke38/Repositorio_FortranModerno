# Resolução: Turbulência: Simulação Numérica Direta (DNS)

## Exercícios do Livro

    -  Derive a lei de potência $-5/3$ de Kolmogorov a partir de argumentos de escala.
    -  Mostre que o projetor $P_{ij}(\mathbf{k}) = \delta_{ij} - k_i k_j / k^2$ garante a incompressibilidade.
    -  Analise a estabilidade do esquema de integração de fator integrante para o termo viscoso.
    -  Desenvolva a rotina de desaliasing usando a regra dos 2/3.
    -  Integre a FFTW3 em um programa Fortran para calcular o espectro de energia $E(k)$.
    -  Verifique a taxa de decaimento da energia total para turbulência livre de forças.
    -  Estude a intermitência através do cálculo da curtose dos gradientes de velocidade.
    -  Investigue como modelos de LES (\textit{Large Eddy Simulation}) se comparam à DNS para altos números de Reynolds.
