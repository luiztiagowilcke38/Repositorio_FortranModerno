# Resolução: Mecânica Orbital e Perturbações

## Exercícios do Livro

    -  Por que a perturbação J2 é útil para o design de órbitas Heliossíncronas?
    -  Explique o conceito de Esfera de Hill e como ela define a estabilidade de uma lua.
    -  Desenvolva um solver RK4 para integrar uma órbita de transferência de Hohmann.
    -  Calcule o tempo de vida orbital de um satélite em LEO sujeito ao arraste atmosférico.
    -  Verifique a conservação da energia e do momento angular em órbitas puramente keplerianas.
    -  Simule o problema dos Três Corpos Restrito e plote as superfícies de Hill (Pontos de Lagrange).
    -  Como o software `STK` ou bibliotecas da NASA utilizam Fortran para garantir precisão centimétrica em missões interplanetárias?
