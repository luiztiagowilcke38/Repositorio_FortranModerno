# Resolução: Sincronização e o Modelo de Kuramoto

## Exercícios do Livro

    -  Derive o limite de acoplamento crítico $K_c = 2/(\pi g(0))$ para uma distribuição Lorentziana de frequências.
    -  Mostre que as equações de Kuramoto possuem simetria de rotação global.
    -  Explique o conceito de estados "quimera" (\textit{Chimera states}).
    -  Adicione ruído estocástico (termo de Langevin) no solver de fases.
    -  Implemente o acoplamento em redes de mundo pequeno (\textit{Small-world}).
    -  Verifique a histerese na transição de sincronização para distribuições de frequência não-imodais.
    -  Estude a dependência de $r$ com $K/K_c$.
    -  Investigue como o modelo de Kuramoto é aplicado no estudo de ondas cerebrais (EEG).
