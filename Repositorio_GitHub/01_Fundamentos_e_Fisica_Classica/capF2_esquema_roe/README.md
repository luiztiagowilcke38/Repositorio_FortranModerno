# Resolução: Aerodinâmica Compressível e Captura de Choques

## Exercícios do Livro

    -  Prove que as médias de Roe satisfazem a propriedade $\mathbf{F}_R - \mathbf{F}_L = A_{roe}(\mathbf{U}_R - \mathbf{U}_L)$.
    -  Explique a necessidade da correção de entropia (Harten-Hyman) para evitar choques não-físicos.
    -  Adicione limitadores de fluxo (flux limiters) como o MinMod para atingir alta ordem de precisão (TVD).
    -  Simule o problema de interação choque-onda (Shu-Osher) e verifique a resolução de estruturas finas.
    -  Compare o esquema de Roe com o esquema de Rusanov (Lax-Friedrichs Local) em termos de difusividade numérica.
