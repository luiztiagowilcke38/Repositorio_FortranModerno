# Resolução: Atratores Estranhos e Expoentes de Lyapunov

## Exercícios do Livro

    -  Derive a contração de volume do sistema de Lorenz: $\nabla \cdot \dot{\mathbf{x}} = -(\sigma + 1 + \beta)$.
    -  Mostre que o sistema de Rössler possui apenas uma não-linearidade.
    -  Explique por que o teorema de Poincaré-Bendixson proíbe o caos em sistemas 2D.
    -  Desenvolva um solver RK4 para visualizar o atrator de Lorenz no plano $x-z$.
    -  Implemente a rotina de renormalização para o cálculo do espectro completo de Lyapunov.
    -  Estude a transição para o caos via duplicação de período no mapa logístico.
    -  Verifique como o horizonte de previsibilidade $T_h \sim 1/\lambda_L$ varia com o ruído numérico.
    -  Investigue o uso de reservatórios neurais (\textit{Reservoir Computing}) para prever sistemas caóticos.
