# Resolução: Fortran Moderno: Uma Visão Geral

## Exercícios do Livro

  - Instrução de identificação (`PROGRAM nome`);
  - Cláusulas `USE` para importação de módulos;
  - `IMPLICIT NONE` (obrigatório em código de qualidade);
  - Declarações de variáveis;
  - Instruções executáveis;
  - `CONTAINS` (opcional) seguido de subprogramas internos;
  - `END PROGRAM nome`.
