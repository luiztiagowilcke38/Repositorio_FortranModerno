# Resolução: Escoamentos Multifásicos: Volume of Fluid (VOF)

## Exercícios do Livro

    -  Explique por que a conservação estrita de $C$ é crucial para evitar perda de massa.
    -  Derive a força de tensão superficial via formulação \textit{Continuum Surface Force} (CSF).
    -  Discuta a diferença entre métodos \textit{Interface Tracking} (Lagrangeano) e \textit{Interface Capturing} (VOF/Level-set).
    -  Codifique a rotina de reconstrução PLIC para interfaces 2D.
    -  Adicione o termo de tensão superficial de Laplace no solver de Navier-Stokes.
    -  Simule o teste de advecção de Zalesak (disco entalhado) para verificar a difusão do VOF.
    -  Estude a ascensão de uma bolha de ar na água considerando a flutuabilidade.
    -  Investigue o método acoplado \textit{CLSVOF} que combina VOF e Level-set para máxima precisão.
