## Exercícios do Livro
 -  Derive a Regra de Simpson 1/3 integrando o polinômio interpolador de Lagrange que passa por $(x_0, f_0), (x_1, f_1), (x_2, f_2)$.
     -  Use a Quadratura de Gauss-Legendre com 5 pontos para calcular $\int_0^1 e^{-x^2} \, dx$ e compare com o valor exato via função erro (`ERF`).
     -  Implemente a Integração de Monte Carlo para calcular o volume de uma hiperesfera de dimensão $d=10$. Compare com a fórmula analítica $V_d(R) = \frac{\pi^{d/2}}{\Gamma(d/2+1)}R^d$.
