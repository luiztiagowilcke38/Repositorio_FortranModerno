# Resolução: Convecção de Rayleigh-Bénard: Ordem e Caos nos Fluidos

## Exercícios do Livro

    -  Derive a condição crítica de instabilidade linear para uma camada infinita de fluido entre placas rígidas.
    -  O que representam os números adimensionais de Rayleigh ($Ra$) e Prandtl ($Pr$)?
    -  Desenvolva um solver de diferenças finitas 2D para a equação de calor com o termo advectivo de convecção.
    -  Visualize os vetores de velocidade das células de convecção formadas.
    -  Investigue a transição de calor (Nusselt number) em função do Número de Rayleigh.
    -  Simule o sistema de convecção com condições de contorno de fluxo constante vs temperatura constante.
    -  Como a convecção no manto terrestre (Capítulo 92) difere da convecção atmosférica clássica devido à reologia do magma?
