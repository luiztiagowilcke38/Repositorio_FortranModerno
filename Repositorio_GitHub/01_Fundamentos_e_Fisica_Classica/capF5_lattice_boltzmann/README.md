# Resolução: Mecânica de Fluidos Estatística: O Método de Lattice Boltzmann

## Exercícios do Livro

    -  Derive a equação de Navier-Stokes a partir da equação de Lattice Boltzmann usando a expansão de Chapman-Enskog.
    -  Demonstre por que a viscosidade cinemática é dada por $\nu = c_s^2(\tau - 1/2)\Delta t$.
    -  Implemente condições de contorno de não-escorregamento (Bounce-back) para paredes curvas.
    -  Compare o LBM com métodos de volumes finitos na simulação de escoamento em meios porosos.
    -  Meça a largura de banda de memória (Memory Bandwidth) utilizada pela SuperDiFF durante o passo de streaming.
