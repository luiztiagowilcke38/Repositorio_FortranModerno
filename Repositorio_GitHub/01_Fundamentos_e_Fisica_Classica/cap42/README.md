# Resolução: Fluidodinâmica Geofísica: Atmosfera e Oceano

## Exercícios do Livro

    -  Derive a relação de dispersão para ondas de Rossby planárias em um plano-$\beta$.
    -  Mostre que a vorticidade potencial é conservada identicamente no modelo de águas rasas.
    -  Explique o fenômeno da intensificação de borda oeste (Corrente do Golfo) via balanço de vorticidade.
    -  Adapte o solver para incluir o efeito-$\beta$ ($f = f_0 + \beta y$).
    -  Adicione um termo de vento (\textit{wind stress}) na camada superficial.
    -  Simule o ajuste geostrófico de uma perturbação inicial de pressão.
    -  Verifique a formação de ondas de Kelvin em uma costa sólida.
    -  Investigue como as equações primitivas 3D estendem os resultados do modelo de águas rasas.
