# Resolução: Expoentes de Lyapunov e Quantificação do Caos

## Exercícios do Livro

    -  Explique a relação entre a soma de todos os expoentes de Lyapunov e a taxa de contração de volume no espaço de fase.
    -  O que significa um sistema ter $\lambda_{max} = 0$?
    -  Calcule o $\lambda_{max}$ para o sistema de Lorenz com $\rho=28$ e verifique se ele é positivo.
    -  Desenvolva um script para gerar um diagrama de bifurcação para o Mapa Logístico $x_{n+1} = r x_n (1-x_n)$.
    -  Estude como o erro na estimativa de $\lambda$ decai com o número de passos $N$.
    -  Verifique a transição do caos para janelas de periodicidade no mapa logístico variando o parâmetro $r$.
    -  Como a análise de séries temporais de Lyapunov é usada para prever instabilidades em redes elétricas e mercados financeiros?
