# Resolução: Turbulência em Escoamentos: Modelos RANS e Fechamento

## Exercícios do Livro

    -  Derive a equação de transporte para a energia cinética turbulenta $k$.
    -  Discuta a validade da hipótese de Boussinesq para escoamentos com alta rotação.
    -  Implemente o modelo de turbulência de uma equação de Spalart-Allmaras.
    -  Analise o perfil de velocidade em uma camada de limite e compare com a "Lei da Parede".
    -  Use a SuperDiFF para simular o escoamento sobre um aerofólio NACA 0012 e calcule o coeficiente de arrasto turbulento.
