# Resolução: Propulsão a Jato e Ciclos de Brayton

## Exercícios do Livro

    -  Por que a turbina de um motor a jato é o componente mais crítico em termos de limites de temperatura?
    -  Explique o conceito de empuxo específico e consumo específico de combustível (SFC).
    -  Desenvolva um código que calcule o empuxo líquido para um motor operando em diferentes números de Mach (0.0 a 2.0).
    -  Adicione a modelagem de um \textit{Afterburner} (Pós-combustor) e calcule o ganho de empuxo.
    -  Otimize a razão de pressão para maximizar o trabalho útil de um ciclo com regeneração.
    -  Estude como a variação de $\gamma$ com a temperatura na câmara de combustão afeta os resultados.
    -  Como a simulação de combustão turbulenta em Fortran permitiu o desenvolvimento de motores \textit{Lean Burn} ultra-eficientes?
