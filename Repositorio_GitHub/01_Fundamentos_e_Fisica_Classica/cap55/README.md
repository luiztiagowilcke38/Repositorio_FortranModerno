# Resolução: Geometria Fractal e Conjuntos de Mandelbrot

## Exercícios do Livro

    -  Prove que o conjunto de Mandelbrot é contido em um disco de raio 2.
    -  Derive a dimensão fractal da Curva de Koch e do Conjunto de Cantor.
    -  Explique a relação entre o conjunto de Mandelbrot e os conjuntos de Julia.
    -  Estenda o código para gerar o conjunto de Julia para um $c$ fixo.
    -  Implemente um algoritmo de \textit{Smooth Coloring} para evitar bandas de cores artificiais.
    -  Verifique a auto-similaridade realizando zooms em ordens de magnitude $10^{-10}$.
    -  Estude a distribuição do tempo de escape próximo à fronteira do conjunto.
    -  Investigue como fractais são usados para modelar o crescimento de colônias de bactérias (modelo DLA).
