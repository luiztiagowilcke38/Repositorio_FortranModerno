## Exercícios do Livro

    -  Derive a velocidade das ondas de Alfvén a partir das equações de momentum linearizadas.
    -  Demonstre o teorema do "congelamento do fluxo" para plasmas ideais.
    -  Implemente o esquema de limpeza de divergência (Divergence Cleaning) de Dedner.
    -  Simule o problema da folha de corrente (Current Sheet) e observe a reconexão magnética.
    -  Use a SuperDiFF para investigar turbulência magnetohidrodinâmica em ambientes interestelares.
